# CSS Box Model
CSS Box Model ek box hai jo har HTML element ko wrap karta hai. Isme 4 parts hote hain:
1. **Margin:** Border ke bahar ki space.
2. **Border:** Content aur Padding ke bahar ki line.
3. **Padding:** Content aur Border ke beech ki space.
4. **Content:** Asli text ya image.

```css
div {
  width: 300px;
  padding: 50px;
  border: 15px solid green;
  margin: 20px;
}
```
