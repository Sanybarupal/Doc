# HTML Kya Hai? (Introduction to HTML)

HTML ka full form hai **HyperText Markup Language**. Ye kisi bhi website ka basic structure banane ke kaam aati hai. 

Jaise ek ghar banane ke liye eent (bricks) ki zarurat hoti hai, waise hi ek website banane ke liye HTML zaroori hai.

## HTML Ka Use Kyun Hota Hai?
- Web pages ka skeleton tayar karne ke liye.
- Images, Text, aur Links ko webpage par show karne ke liye.
- Browser ko batane ke liye ki kaunsa content heading hai, aur kaunsa paragraph.

## Basic HTML Structure

Har HTML page me ye basic code zaroori hota hai:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Mera Pehla Webpage</title>
</head>
<body>
    <h1>Namaste World!</h1>
    <p>Ye mera pehla paragraph hai.</p>
</body>
</html>
```

## Elements & Tags
HTML tags hamesha angle brackets (`< >`) mein likhe jaate hain. Zyada tar tags pair mein aate hain (opening tag aur closing tag).

| Tag | Use |
|---|---|
| `<h1>` se `<h6>` | Headings banane ke liye |
| `<p>` | Paragraph ke liye |
| `<a href="...">` | Links banane ke liye |
| `<img src="...">` | Image lagane ke liye |

> [!TIP]
> Hamesha yaad rakhein ki apne closing tags properly close karein, jaise `</p>`. Warna layout kharab ho sakta hai!
