# Git Stash

Maan lo aap kisi file par kaam kar rahe ho, aur achanak boss bolte hain doosri branch mein ek urgent bug fix karna hai. Par aapka kaam abhi adha hai aur aap use commit nahi karna chahte. 

Tab aap `git stash` ka use karte hain. Ye aapke uncommitted changes ko temporary chhupa kar rakh leta hai aur working directory clean kar deta hai.

## Examples
Chhupana:
```bash
git stash
```

Wapas lana:
```bash
git stash pop
```
