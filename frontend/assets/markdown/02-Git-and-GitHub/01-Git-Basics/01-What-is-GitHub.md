# GitHub Kya Hai? (What is GitHub?)

GitHub ek cloud-based platform hai jahan developers apna code store, manage aur share kar sakte hain. Yeh Git version control system ka use karta hai.

Maan lijiye **Git** ek camera hai jo aapke code ki photo (versions) khichta hai, toh **GitHub** Instagram hai jahan aap un photos ko duniya ke sath share karte ho!

## GitHub Ke Fayde:
1. **Collaboration:** Duniya bhar ke developers ek hi project par ek sath kaam kar sakte hain.
2. **Backup:** Agar aapka laptop kharab ho jaye, toh code cloud par safe rehta hai.
3. **Open Source:** Aap dusron ke projects dekh sakte hain aur unme contribute kar sakte hain.
4. **Portfolio:** Aapka GitHub profile aapke resume ki tarah kaam karta hai.
