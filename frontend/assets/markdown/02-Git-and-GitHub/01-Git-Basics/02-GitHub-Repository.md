# GitHub Repository Kya Hai?

Ek **Repository** (ya "repo") bilkul ek folder ki tarah hoti hai jahan aapka poora project rakha jata hai. Isme aapke project ke saare files, images, aur coding history shamil hoti hai.

## Repo Ke Prakar (Types):
1. **Public Repo:** Isey internet par koi bhi dekh sakta hai. Ye open-source projects ke liye best hai.
2. **Private Repo:** Isey sirf aap aur aapke invite kiye hue team members hi dekh sakte hain.

## Local vs Remote Repo
- **Local Repo:** Jo repo aapke computer mein saved hai.
- **Remote Repo:** Jo repo GitHub server par hosted hai.

Hamesha yaad rakhein: Aap local repo mein changes karte hain aur fir unhe remote repo par 'push' karte hain.
