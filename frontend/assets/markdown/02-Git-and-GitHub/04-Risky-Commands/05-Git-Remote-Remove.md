# Remote Remove Origin

Command: `git remote remove origin`

## Ye kya karta hai?
Yeh aapke local project aur GitHub server ke beech ka link tod deta hai. 

> [!WARNING]
> Isse aapka code delete toh nahi hoga, par aap aage se GitHub par code push ya pull nahi kar payenge jab tak aap naya remote add nahi karte.
