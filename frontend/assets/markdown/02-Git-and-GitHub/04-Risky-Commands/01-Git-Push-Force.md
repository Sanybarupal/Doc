# Git Push Force

Command: `git push --force` ya `git push -f`

## Ye kya karta hai?
Ye command zabardasti (forcefully) aapke local commits ko GitHub server par bhej deta hai, bhale hi server par doosre naye changes kyu na hon. Ye server ki history ko overwrite kar deta hai.

> [!CAUTION]
> **Risky Kyun Hai?** Agar aapki team ke kisi aur developer ne code push kiya tha, aur aapne force push kar diya, toh us developer ka code permanently delete ho jayega server se! Ise tabhi use karein jab aap 100% sure hon.
