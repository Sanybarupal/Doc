# Git Commit

`git commit` command aapke staged changes ka ek snapshot (photo) le leti hai aur usko history mein hamesha ke liye save kar leti hai. Har commit ka ek message hona zaroori hai.

## Example
```bash
git commit -m "Navigation bar add kar diya"
```

> [!TIP]
> Hamesha clear aur samajh aane wale commit messages likhein taaki baad mein history padhna asaan ho.
