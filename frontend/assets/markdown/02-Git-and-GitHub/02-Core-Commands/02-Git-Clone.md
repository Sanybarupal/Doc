# Git Clone

`git clone` ka use internet (GitHub, GitLab) par majood kisi repository ki exact copy apne computer mein download karne ke liye hota hai.

## Example
Kisi existing project ko apne PC par laane ke liye:

```bash
git clone https://github.com/username/project-name.git
```

Yeh command automatically `git init` kar deti hai, saara data download karti hai, aur history track karti hai.
