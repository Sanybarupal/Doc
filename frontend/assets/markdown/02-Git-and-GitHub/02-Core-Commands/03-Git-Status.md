# Git Status

`git status` sabse zyada use hone wali command hai. Yeh aapko batati hai ki aapke project ka current haal kya hai.

Isse aapko pata chalta hai ki:
- Kaunsi files modify hui hain.
- Kaunsi files stage ho chuki hain (commit hone ke liye taiyar).
- Kaunsi files untracked hain.

## Example
```bash
git status
```
