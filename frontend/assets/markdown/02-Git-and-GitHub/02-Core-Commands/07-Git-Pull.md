# Git Pull

`git pull` command GitHub se naye changes (agar kisi aur developer ne kiye hain) aapke local computer mein download karke aapke current code mein automatically mix (merge) kar deti hai.

## Example
```bash
git pull origin main
```

> [!WARNING]
> Agar aapne same file mein kaam kiya hai jisme naye changes aaye hain, toh "Merge Conflict" aa sakta hai. Isko manually fix karna padta hai.
