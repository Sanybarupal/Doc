# Git Add

`git add` command aapke changes ko "Staging Area" mein bhejti hai. Staging area ek waiting room ki tarah hai, jahan aap un files ko rakhte hain jinhe aap aagle commit mein save karna chahte hain.

## Example

Ek specific file ko stage karne ke liye:
```bash
git add index.html
```

Saari modified files ko ek sath stage karne ke liye:
```bash
git add .
```
