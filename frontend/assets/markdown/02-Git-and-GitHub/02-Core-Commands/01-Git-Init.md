# Git Init

`git init` ka use ek naye Git repository ko shuru (initialize) karne ke liye hota hai. Yeh command ek normal folder ko Git folder mein badal deta hai jismein ek hidden `.git` folder create ho jata hai.

## Example
Aapne ek naya project folder banaya, ab usme Git start karna hai:

```bash
cd mera-project
git init
```

> [!NOTE]
> Ek baar project mein `git init` chalane ke baad hi aap dusri Git commands (jaise add, commit) use kar sakte hain.
